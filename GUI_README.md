# BlazeNeuro GUI - Iron Man Style Interface

## Features

### Terminal Interface (JARVIS)
- Futuristic ASCII art banner
- Animated boot sequence
- Color-coded system status
- Interactive command center
- System monitor, network scanner, file explorer

### Web GUI
- Cyberpunk/hacker aesthetic
- Real-time system monitoring with animated bars
- Interactive terminal with custom commands
- Network visualization with animated nodes
- File browser
- Glitch effects and scanline overlay
- Matrix-style animations

## Installation

After building BlazeNeuro system:

```bash
sudo ./gui/install.sh
```

## Usage

### Terminal Interface
```bash
blazeneuro
```

### Web Interface
From terminal menu, select option 5, then open browser:
```
http://localhost:8080
```

## Terminal Commands
- `help` - Show available commands
- `clear` - Clear terminal
- `status` - System status
- `neofetch` - System info
- `hack` - Neural hack simulation
- `matrix` - Easter egg

## Customization

Edit files in `gui/` directory:
- `terminal/jarvis.sh` - Terminal interface
- `web/style.css` - Colors and styling
- `web/script.js` - Animations and behavior

## Theme
- Primary: Neon Green (#0f0)
- Secondary: Cyan (#0ff)
- Background: Black with green tint
- Effects: Scanlines, glitch, glow
