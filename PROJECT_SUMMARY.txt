================================================================================
                            BUILD STAGES
================================================================================

Stage 1: Environment Preparation (5 min)
  - Create LFS directory structure
  - Setup lfs user
  - Copy source packages

Stage 2: Cross-Toolchain (60-90 min)
  - Binutils 2.42
  - GCC 13.2.0 (Pass 1 & 2)
  - Linux headers 6.7.4
  - Glibc 2.39

Stage 3: Temporary System (20-30 min)
  - Bash, Coreutils, Findutils
  - Grep, Gzip, Tar, XZ

Stage 4: Final System (15-20 min)
  - util-linux, e2fsprogs
  - Chroot environment

Stage 5: Configuration (30-45 min)
  - Linux kernel compilation
  - GRUB bootloader
  - System configuration

Stage 6: GUI (Optional)
  - Placeholder for X11/GTK

Total Build Time: 2-4 hours (local), 1.5-2 hours (GitHub Actions)

================================================================================
                         USB PERSISTENCE
================================================================================

Partition Layout:
  /dev/sdX1  →  512MB FAT32  →  /boot (GRUB + Kernel)
  /dev/sdX2  →  Remaining EXT4  →  / (Root, Read-Write)

Persistence Method:
  - Direct ext4 filesystem (no overlay)
  - All changes saved immediately
  - Full journaling for data integrity
  - No performance overhead

================================================================================
                        TROUBLESHOOTING
================================================================================

Build Fails:
  1. Check logs/build.log
  2. Verify host requirements: ./verify.sh
  3. Retry specific stage: sudo ./build.sh stage2
  4. Clean and rebuild: sudo ./build.sh clean

USB Won't Boot:
  1. Check BIOS/UEFI boot order
  2. Verify device: lsblk
  3. Recreate USB: sudo ./build.sh usb /dev/sdX

Insufficient Space:
  1. Check space: df -h /mnt/lfs
  2. Clean build: sudo ./build.sh clean
  3. Remove old sources
