# BlazeNeuro Native GUI

Custom desktop environment using X11 and GTK3 in C.

## Components
- **BlazeWM**: X11 window manager
- **Panel**: Top bar with clock
- **Launcher**: Application grid

## Build
```bash
cd gui-native
make
sudo make install
```

## Technology
- X11 (like Ubuntu)
- GTK3 (like GNOME)
- C language
- Native compiled
